## 業務知識

### 主要業務流程

1. **接收客戶訂單**（Purchase Order）— 通常為 PDF 或 Excel
2. **產品比對** — 將客戶 Part# 對應 ERP 中的品項
3. **採購歷史查詢** — 查看是否曾採購、最近進貨價
4. **報價試算** — 考慮成本、運費、毛利率
5. **報價回覆** — 產出客戶報價單（Beck A05 格式）

### 產品分類（kino 代碼）

| 代碼 | 分類 | 說明 |
|------|------|------|
| AC | 冷氣 | 壓縮機、冷凝器、蒸發器等 |
| AH | 底盤 | 避震器、三角架、拉桿等 |
| AP | 引擎 | 汽門、正時、水泵等 |
| AE | 電裝 | 啟動馬達、發電機、點火線圈 |
| AB | 煞車 | 來令片、碟盤、煞車分泵 |
| AF | 燃油 | 噴油嘴、汽油泵、EGR 閥 |
| AW | 雨刷 | 雨刷片、雨刷臂 |
| AG | 墊片 | 引擎墊片、氣缸床墊片 |

### 常見客戶 OE 號格式

- **Federal-Mogul/Beck**: SAP PO#（如 4503218135），Buyer Part#（如 UXC0460146）
- **OE 號**: 原廠零件號碼（如 8430702220），各車廠格式不同
- **SMP 號**: Standard Motor Products 編號
- **UXC 碼**: 跨供應商標準編號，同品項不同供應商共用同一 UXC 碼

### 報價公式

```
建議報價 = (進貨成本 + 每件運費分攤 + 每件測試費分攤) / (1 - 目標毛利率)
```

- 運費分攤 = 總運費 / 訂單品項數
- 常見目標毛利率：20%~30%
- 幣別通常為 USD（美金）
